# Stopio-ultimate-userbot26
🚀 Multi-Functional Userbot V2 (Pyrogram)Userbot Telegram berbasis Pyrogram V2 yang dilengkapi dengan fitur Asisten Bot, Animasi, Game, Lokasi Palsu, dan integrasi AI (GPT-4). Ringan, stabil, dan siap di-deploy ke Railway.🛠 Persiapan Bahan (BACA DULU!)Sebelum melakukan deploy, lu wajib punya 4 "kunci" utama ini:1. Ambil API ID & API HASHBuka my.telegram.org.Masukkan nomor HP lu (pake format internasional: +628...).Masukkan kode verifikasi yang masuk ke chat Telegram lu.Klik API Development Tools.Isi App title dan Short name bebas (contoh: BotGw).Klik Create application.Lu bakal dapet App api_id dan App api_hash. Simpan!2. Ambil BOT TOKEN (Untuk Asisten)Chat @BotFather di Telegram.Ketik /newbot.Kasih nama bot bebas, lalu kasih username (akhiri dengan _bot).Lu bakal dapet HTTP API Token. Simpan sebagai BOT_TOKEN.3. Ambil STRING SESSION (Kunci Akun Utama)Ini buat login ke akun utama lu tanpa masukin kode terus-menerus.Lu bisa pake String Session Generator (biasanya pake Replit atau script lokal).PENTING: Jangan kasih kode String Session ini ke SIAPAPUN, karena ini akses penuh ke akun lu.🚀 Panduan Deploy ke RailwaySetelah bahan lengkap, ikuti langkah ini:Langkah 1: Push ke GitHubBuat repository baru di GitHub (Private disarankan).Upload file main.py, requirements.txt, dan .env.Langkah 2: Koneksi ke RailwayLogin ke Railway.app.Klik + New Project > Deploy from GitHub repo.Pilih repository userbot lu.Langkah 3: Setting Variables (WAJIB!)Di dashboard Railway, masuk ke tab Variables dan masukkan data berikut satu per satu:API_ID: (Isi dengan API ID lu)API_HASH: (Isi dengan API HASH lu)STRING_SESSION: (Isi dengan String Session lu)BOT_TOKEN: (Isi dengan Token Bot dari BotFather)Langkah 4: Deploy Berhasil!Railway bakal otomatis melakukan building dan deploying.Cek tab Logs. Kalau ada tulisan ✅ SEMUA ONLINE!, berarti bot lu udah jalan.📜 Daftar Perintah (Commands)👤 Akun Utama (Gunakan awalan titik .)PerintahFungsi.pingCek status bot online.uncastHapus 50 pesan terakhir lu (Privasi).lokasi [nama]Kirim lokasi palsu (Share Loc).emAnimasi kurir paket jalan.fake [type]Status Mengetik Abadi (typing/playing/off).slot / .daduFitur game/judi interaktif🤖 Asisten Bot (Gunakan awalan garing /)PerintahFungsi/startMemulai bot asisten/helpLihat daftar panduan lengkap/idCek ID Telegram & Chat ID/tanya [teks]Tanya AI GPT-4 lewat bot⚠️ Catatan PentingKemanan: Jangan pernah membagikan file .env atau nilai STRING_SESSION ke publik.Limit: Gunakan fitur .uncast dan .fake dengan bijak agar akun tidak terkena Flood Limit dari Telegram.Build with ❤️ by StoopioUserbot26.
